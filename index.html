調  Key

短調は和声的短音階（導音を自動的に半音上げる）を基準に度数計算します。

低音進行の入力  Bass Line Input

#音名変化Oct音価数字（figures）

数字の書式例：無し(5/3の三和音) / 6 / 6/4 / 7 / 6/5 / 4/3 / 4/2（または2） / 9 / 7/9 / 4-3（掛留）/ 7-6 / 9-8 / 単独の変化記号 # b（第3音の変化）/ 増六和音 it6 fr6 ger6。
それ以外の任意の組み合わせ（例：#4, b7, 2 4 6）も度数として直接解釈されます。

低音進行の配置ルール：2番目以降の低音は、直前の低音との音程を増5度（8半音）以内にしてください（1番目の低音は加線にはみ出しても構いません）。 違反すると「五線譜を生成」時にエラーが表示されます。
短調でVの和音（属和音）を入力する行では、数字欄の下に導音の扱い（♯3にする／自然短音階のまま♭3にする）を選ぶプルダウンが自動で表示されます。

大譜表  Score

高音部の点＝その和音構成音に一致する全てのピッチ（オクターブ4〜5の範囲） 低音部＝入力された低音音符
